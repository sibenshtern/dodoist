# Архитектура

Dodoist состоит из трёх контейнеров: Angular SPA в браузере, Django REST API на сервере и PostgreSQL как единственное хранилище данных.

Angular SPA работает в браузере и общается с бэкендом только через REST API. Django REST API — без состояния: принимает запросы, проверяет JWT-токен, выполняет бизнес-логику, пишет в PostgreSQL. Горизонтально масштабируется без дополнительных усилий, потому что сессии хранятся в базе, не в памяти процесса. PostgreSQL — единственная база данных: пользователи, проекты, задачи, история изменений, UUID-первичные ключи, JSONB для описаний задач, float8 для порядка при drag-and-drop.

## Коммуникация

Браузер общается с бэкендом по HTTPS. Аутентификация — Bearer JWT. Access-токен живёт 15 минут, refresh-токен — 30 дней. Явный выход инвалидирует сессию в базе.

Пароли передаются через HTTP Basic Auth при входе и через заголовок `X-Password` при регистрации — чтобы не попадать в логи приложений, которые пишут тела запросов.
